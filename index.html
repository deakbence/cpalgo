Iteratív algoritmus

Az iteratív algoritmus a kitevő bináris felírásán alapszik. Fenntartunk egy segédváltozót, ami az alap aktuális bitnek megfelelő hatványát tárolja, és ha ez a bit aktív, az eredményt megszorozzuk a segédváltozóval. Például: 

int fpow(int b, int p) {
   int res = 1;
   int pow = b;
   for(; p; p >>= 1) {
      if(p & 1) {
         res *= pow;
      }
      pow *= pow;
   }
   return res;
}
